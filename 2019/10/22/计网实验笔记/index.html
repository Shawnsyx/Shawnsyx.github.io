计网笔记

lab1

 在计网实验中遇到了些小问题，记录下来。

StartProxy Exception

 原因是利用Java开启一个进程后再开启相同进程会抛出这个异常，其中eclipse会提示

Address already in use: NET_Bind

 于是上网查询，发现原因是开启服务器后尽管关闭了eclipse，进程仍然没有被终结。解决办法：

  • netstat -p tcp -ano 查看端口被哪个进程占用
  • taskkill -PID [进程ID] -F 强制终止进程

 成功解决。
